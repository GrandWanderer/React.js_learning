# Travel Destinations

Static React/Vite application for Laboratory Work No. 6 on the topic "useEffect and localStorage".

The project demonstrates:
- `useState` for storing the destination list on the top level
- `useEffect` for loading and saving data
- `map()` for rendering destination cards
- `filter()` for switching between all, visited, and planned modes
- conditional rendering for the empty state
- a controlled form with `onChange` and `onSubmit`
- localStorage persistence for added destinations and visited status

After reloading the page, user-added destinations and visited/planned changes remain saved in the browser.

Run locally:

```bash
npm install
npm run dev
npm run build
```
