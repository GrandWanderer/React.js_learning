# Travel Destinations

Static React/Vite application for Laboratory Work No. 4 on the topic "List Rendering and Conditional Rendering".

The project demonstrates:
- `useState` for storing the destination list on the top level
- `map()` for rendering destination cards
- `filter()` for switching between all, visited, and planned modes
- conditional rendering for the empty state
- toggling the `visited` status for each destination card

Run locally:

```bash
npm install
npm run dev
npm run build
```
