# Travel Destinations

Static React/Vite application for Laboratory Work No. 3 on the topic "Local Component State Management".

The project demonstrates:
- `useState` for storing the destination list on the top level
- toggling the `visited` status for each destination card
- passing data and callback functions through the component hierarchy
- updating the visited counter in the header reactively

Run locally:

```bash
npm install
npm run dev
npm run build
```
