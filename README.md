# Travel Destinations

Static React/Vite application for Laboratory Work No. 5 on the topic "Event Handling and Controlled Forms".

The project demonstrates:
- `useState` for storing the destination list on the top level
- `map()` for rendering destination cards
- `filter()` for switching between all, visited, and planned modes
- conditional rendering for the empty state
- a controlled form with `onChange` and `onSubmit`
- adding new destination cards without page reload

Run locally:

```bash
npm install
npm run dev
npm run build
```
